
Semantic Web Technology nao tem haver com modelo de dados mas com dados. 
Dados sao o que alimentam a semantic web e sparql auxilia nesse processo. 


O autor define semantic web (no conceito dele) como 
	
	a set of standards and best practices for sharing data 
	and the semantics of that data over the Web for use 
	by applications. 
	

A set of standards... 
	o W3C criou alguns stardards como: 
		RDF data model
		SPARQL
		RDFSchema
		OWL : para vocabularios e ontologias

best practices for sharing data... over the Web for use by applications

	Aqui entra o conceito de screen scraping (varrer um pagina humam readable html 
	para obter informações). Denindo um padrão que maquinas possam ler e entender 
	(usando URIs), evita esse tipo de varredura.
